### Extra type usage example:

```gdscript
var health = 5

func _ready():
	Console.add_command('set_health', self, 'health')\
		.set_description('Set health')\
		.add_argument('health', Console.FloatRangeType.new(0, 5, 0.5))\
		.register()
```
